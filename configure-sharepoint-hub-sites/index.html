Configure SharePoint Hub Sites

By admin

March 30, 2018

The most awaiting feature “Hub Sites” is released now. Hub sites allow the related site content to roll up to the Hub site’s home page, like News. There is no specific template for Hub sites and you can convert your existing modern Team site or Communication site to a Hub site by running PowerShell script. You can have up to 50 Hub sites in your tenant as per MS article.

We can configure the Hub sites in 4 steps.

Converting an existing Team site or Communication site to Hub Site

  1. 1. Open SharePoint online PowerShell (Download the latest version)
    _ Connect-SPOService “https://abcd-admin.sharepoint.com"   _ Enter Tenant admin user name and password to login
    1. Register existing Team site or Communication site as Hub site     Register-SPOHubSite “https://abced.sharepoint.com/sites/modernteamsite";

Hub site settings

Once the above scripts are run, you can see that the “Hub settings” menu item is added to the Gear icon as shown below.

Click on ‘Hub settings’ to change the Hub site Icon and the Display name

Associating a site to the Hub site

The next step is to add Modern team or Communication site to the Hub site.

  1. Browse the site which needs to be added to the Hub site
  2. Clik on gear icon and Site Information

  1. From the drop down ‘Hub site association’ select the Hub site and save the changes.

Configure the Hub site webparts to Roll up the content

The final step is the configure the webparts in the Hub site to show the content from the associated sites.

  1. Navigate to the Hubsite
  2. Edit the home page and then the webpart say ‘News’
  3. In the prperoty panel, select ‘All the sites in the Hub’ option
  4. Save the webpart and publish the page

I observed that it took an hour to get the associated site content to roll up on the hub site home page.
